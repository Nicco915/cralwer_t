### 下一步测试优先级

1. **错误恢复 / 容错测试（优先）**
   - 网络中断：任务拉取/回调时断网、上游 API 5xx、超时。
   - 回调失败：回调接口不可用、重试耗尽、部分成功。
   - 浏览器崩溃：页面渲染异常、Chromium 崩溃、上下文泄漏。
   - 目标：验证节点在异常后能自愈、不丢任务、不重复回调。

2. **长稳运行 / 监控告警（优先）**
   - 长时间运行（数小时）下的内存、句柄、日志、回调成功率。
   - 基础监控指标：任务处理速率、成功率、浏览器重启次数、上游 API 错误率。
   - 告警阈值：成功率低于阈值、连续 N 次拉取失败、内存泄漏趋势。
